# Your-Face
Face Recognition Using OpenCV


## How to Run

Install python-opencv

    $ brew install opencv
    add "export PYTHONPATH=/usr/local/lib/python2.7/site-packages:$PYTHONPATH" to ~/.bashrc or ~/.zshrc
    
    
Run Basic Script

    $ python static.py IMAGE_FILE
    
    $ python live.py
    

## Graphic works 

RGB to Gray

DWT

FFT
